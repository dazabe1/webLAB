## Запуск проекта

Для запуска проекта вам потребуется клонировать данный репозиторий на свой компьютер и запустить соответствующие файлы лабораторных работ. Детальные инструкции по запуску находятся внутри каждой директории.

## Результаты

Конечная цель этих лабораторных работ — разработать полнофункциональный сайт кулинарного магазина с возможностью заказа и доставки продуктов. В результате успешного выполнения всех лабораторных работ вы получите готовый проект, который можно использовать в реальных условиях.
